# Sistema de Gestão de Supermercado

**Aluno:** Pedro Galvão do Amaral Neto

**Disciplina:** Linguagem de Programação II

**Atividade:** Laboratório 1

---

### Descrição

Sistema de gestão de supermercado desenvolvido como atividade prática da disciplina de Linguagem de Programação II. O projeto simula o núcleo de um sistema de caixa e inventário, aplicando os pilares da Orientação a Objetos: **Composição**, **Herança**, **Abstração**, **Polimorfismo** e **Encapsulamento**.
